# Projeto GEIA-25.1

Este é o repositório do projeto **GEIA-25.1**, desenvolvido para gerenciar e processar submissões no diretório `ent-1 marcos`.

## Estrutura do Projeto

-   `/submissions/ent-1 marcos/` - Diretório contendo as submissões organizadas.
-   `📄README.md` - Arquivo de documentação do projeto.

## Como Usar

1. Clone o repositório:
    ```bash
    git clone <url-do-repositorio>
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd GEIA-25.1/submissions/ent-1\ marcos/
    ```
3. Execute o código conforme necessário.
